## Requirements
* node (https://nodejs.org/en/download/)
* nodemon (npm install -g nodemon)
* typescript (npm install -g typescript)
* ts-node (npm install -g ts-node)

## Installation
1. Clone the project
2. Open the terminal
3. Run the command:
~~~
npm install
~~~

## Run Project in Development Environment 
~~~
npm run start-dev
~~~
